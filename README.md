# MX-UI
[3X-UI](https://github.com/MHSanaei/3x-ui) equipped with [Mahsa-Core](https://github.com/GFW-knocker/Xray-core)<br>
<b>MVLESS</b> support (designed for [MahsaNG](https://github.com/GFW-knocker/MahsaNG))<br>
quic support (soon)<br>



## Install on Linux VPS

```bash
bash <(curl -Ls https://raw.githubusercontent.com/GFW-knocker/3x-ui/master/install.sh)
```

## Install on Windows
download & run .exe binary from release<br>
follow readme to open panel on browser
